#+AUTHOR: Yoshiaki Sano
#+LANGUAGE: ja

* 概要

クロスプラットフォーム（macOS/Linux/WSL/Windows）対応の統合開発環境 dotfiles。
Emacs 中心、AI 統合ワークフロー、日本語環境、Zsh + Zinit 構成。

** 主要コンポーネント

| コンポーネント | 規模 | 管理 |
|---|---|---|
| Emacs | ~4,800行, 12モジュール | straight.el + use-package |
| Zsh | ~4,600行, dual-stage | Zinit |
| tmux | ~3,800行, モジュラー | TPM |
| Keyboard | Karabiner/skhd/yabai | OS固有 |

** 設計原則

- *グレースフル劣化*: モダンツール未導入時は自動フォールバック
- *遅延読み込み*: Zinit / use-package で段階的ロード
- *クロスプラットフォーム*: OS 固有機能は条件分岐で対応

* セットアップ

** 前提条件

- Emacs 28+, Zsh 5.8+, Git 2.25+, tmux 2.9+

** インストール

#+begin_src bash
git clone https://github.com/ysano/dotfiles.git ~/dotfiles
cd ~/dotfiles
./link.sh
#+end_src

** OS 固有設定

- *macOS*: =cp -r karabiner ~/.config/=
- *WSL*: =sudo cp wsl/etc_fonts/local.conf /etc/fonts/local.conf=

* カスタマイズ

ローカル設定（git 管理外）:
- =~/.zshrc.local= — Zsh ローカル設定
- =~/.emacs.d/inits/init-local.el= — Emacs ローカル設定
