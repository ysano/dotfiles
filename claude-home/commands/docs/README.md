# Documentation Commands

ソフトウェアドキュメント作成・保守のためのコマンド群。

> **Knowledge Base**: 各コマンドの詳細な手順は `docs` Skill (`skills/docs/`) に集約されている。
> コマンドは Skill への薄いエントリーポイントとして機能する。

## Available Commands

- **create-architecture-documentation.md** - 包括的なアーキテクチャドキュメントを作成
- **generate-api-documentation.md** - API ドキュメントを自動生成
- **create-onboarding-guide.md** - 開発者向けオンボーディングガイドを作成
- **migration-guide.md** - バージョンアップグレード用マイグレーションガイドを作成
- **troubleshooting-guide.md** - 一般的な問題のトラブルシューティングガイドを作成

## Related

- **Skill**: `docs` - ドキュメント種別クイックリファレンス、品質基準、手順詳細
- **Agent**: `docs-pro` - 包括的ドキュメント戦略の専門エージェント
- **Agent**: `api-docs-pro` - OpenAPI / SDK ドキュメント作成の専門エージェント
