## Instructions

プロジェクトのマイルストーン進捗を追跡・分析する。

1. **ツール確認**: Linear MCP、GitHub CLI、git リポジトリへのアクセスを検証
2. **データ収集**: Linear/GitHub からマイルストーン・関連タスクを取得、git タグからリリース履歴を分析
3. **進捗分析**: マイルストーンごとの完了率、速度トレンド、ブロッキング課題、残り時間を算出
4. **予測分析**: バーンダウンレートから完了日を予測、信頼度レベル付き
5. **リスク評価**: スケジュール・スコープ・リソース・依存関係・技術リスクを評価
6. **レポート生成**: タイムライン、進捗指標、リスクヒートマップ、推奨アクションを含むレポートを出力
7. **依存関係マップ**: マイルストーン間・チーム間の依存関係とクリティカルパスを可視化

## 前提条件

- Git リポジトリアクセス
- Linear MCP 接続（推奨）または GitHub マイルストーン/プロジェクトボード
