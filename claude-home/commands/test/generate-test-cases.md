description: "Generate comprehensive test cases automatically"
---

包括的なテストケースを自動生成する。

手順の詳細は `test` Skill の `references/generate-test-cases.md` を参照。

1. `test` Skill の `references/generate-test-cases.md` を Read して手順を確認
2. ターゲットコードを分析 (引数: `$ARGUMENTS`)
3. ポジティブ・ネガティブ・エッジケースを生成
4. 生成テストの実行成功を確認して報告

See also: `/test:write-tests`, `/test:test-coverage`
