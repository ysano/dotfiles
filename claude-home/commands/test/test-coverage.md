テストカバレッジを分析・改善する。

手順の詳細は `test` Skill の `references/test-coverage.md` を参照。

1. `test` Skill の `references/test-coverage.md` を Read して手順を確認
2. 手順に従いカバレッジ分析を実施
3. Line/Branch/Function カバレッジを計測
4. カバレッジギャップを特定して報告

See also: `/test:write-tests`, `/test:add-mutation-testing`
