# Test Commands

テスト・品質保証・テスト自動化のためのコマンド群。コード信頼性の確保と高いテストカバレッジの維持を支援。

> **Knowledge Base**: 各コマンドの詳細な手順は `test` Skill (`skills/test/`) に集約されている。
> コマンドは Skill への薄いエントリーポイントとして機能する。

## Available Commands

- **write-tests.md** - ユニット・統合テストを作成
- **test-coverage.md** - テストカバレッジを分析・改善
- **e2e-setup.md** - E2E テストインフラとフレームワークをセットアップ
- **generate-test-cases.md** - コード分析に基づきテストケースを自動生成
- **setup-comprehensive-testing.md** - 包括的テスト戦略を構築
- **setup-load-testing.md** - 負荷テストを設定しパフォーマンスを計測
- **setup-visual-testing.md** - UI コンポーネントのビジュアルリグレッションテストを実装
- **add-mutation-testing.md** - ミューテーションテストでテストスイート有効性を検証
- **add-property-based-testing.md** - プロパティベーステストでロバストなカバレッジを実装
