## 実行手順

Analyze and explain: **$ARGUMENTS**

1. **High-Level Overview** - コードの目的、解決する問題、システム内での役割を要約
2. **Data Flow** - 入力 → 処理 → 出力のフローを図示
3. **Key Logic** - アルゴリズム、状態管理、エラーハンドリングの核心部分を詳説
4. **Dependencies** - 外部ライブラリ、内部モジュール、API連携を列挙
5. **Potential Issues** - パフォーマンス懸念、セキュリティリスク、改善機会を指摘（求められた場合）
