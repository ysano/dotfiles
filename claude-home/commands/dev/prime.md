## Instructions

Analyze the current project and build comprehensive context:

1. **Codebase Structure** - `git ls-files` でファイル構成を把握、主要ディレクトリの役割を特定
2. **Documentation** - README.md、docs/ を読み込み（存在する場合）
3. **Tech Stack** - 言語、フレームワーク、依存関係、CI/CD 設定を特定
4. **Overview** - プロジェクトの目的、主要コンポーネント、セットアップ手順を2-3文で要約
