# Setup Commands

プロジェクトセットアップ・開発環境構築・インフラ整備のためのコマンド群。

> **Knowledge Base**: 各コマンドの詳細な手順は `setup` Skill (`skills/setup/`) に集約されている。
> コマンドは Skill への薄いエントリーポイントとして機能する。

## Available Commands

- **setup-development-environment.md** - 完全な開発環境をセットアップ
- **migrate-to-typescript.md** - JavaScript プロジェクトを TypeScript に移行
- **setup-monitoring-observability.md** - 監視・オブザーバビリティシステムをセットアップ
- **setup-monorepo.md** - モノレポ構造を構成
- **modernize-deps.md** - プロジェクトの依存関係を更新・近代化
- **setup-formatting.md** - コードフォーマットツールを設定
- **setup-linting.md** - コードリンティングと品質ツールをセットアップ
- **create-database-migrations.md** - データベースマイグレーションを作成・管理
- **design-database-schema.md** - 最適化されたデータベーススキーマを設計
- **design-rest-api.md** - RESTful API アーキテクチャを設計
- **implement-graphql-api.md** - GraphQL API エンドポイントを実装
- **setup-rate-limiting.md** - API レート制限を実装
