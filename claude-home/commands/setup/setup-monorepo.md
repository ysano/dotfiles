モノレポ構造を構成する。

手順の詳細は `setup` Skill の `references/monorepo.md` を参照。

引数でツールを指定可能: `$ARGUMENTS` (nx, lerna, rush, yarn-workspaces, pnpm-workspaces, turborepo)

1. `setup` Skill の `references/monorepo.md` を Read して手順を確認
2. モノレポツールを選定または指定されたツールを使用
3. ワークスペース構造とビルドシステムを構成
4. 検証チェックリストで動作確認
5. 結果を報告

See also: `/setup:setup-development-environment`, `/deploy:ci-setup`
