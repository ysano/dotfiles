RESTful API アーキテクチャを設計する。

手順の詳細は `setup` Skill の `references/rest-api-design.md` を参照。

1. `setup` Skill の `references/rest-api-design.md` を Read して手順を確認
2. API 戦略を策定し、リソース設計
3. 認証・バリデーション・ドキュメントを実装
4. API テストを実施
5. 結果を報告
