# Security Commands

セキュリティ分析・脆弱性検出・堅牢化のためのコマンド群。

> **Knowledge Base**: 各コマンドの詳細な手順は `security` Skill (`skills/security/`) に集約されている。
> コマンドは Skill への薄いエントリーポイントとして機能する。

## Available Commands

- **security-audit.md** - 包括的なセキュリティ監査を実施
- **security-hardening.md** - セキュリティ堅牢化措置を適用
- **dependency-audit.md** - 依存関係の脆弱性を監査
- **add-authentication-system.md** - セキュアな認証・認可システムを実装
