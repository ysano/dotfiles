## 5. 削除コマンド案（マルチOS対応版）

```bash
# バックアップ作成
tar -czf tmux_backup_$(date +%Y%m%d_%H%M%S).tar.gz .tmux/

# テストファイルを削除
find .tmux -type f -path "*/tests/*" -o -path "*/test_*.sh" | xargs rm -f

# 未使用のclaudeステータススクリプトを削除  
rm -f .tmux/scripts/claude-status-*.sh

# 未使用のcoreファイルを削除

# 未使用のClaudeモジュールを削除
rm -f .tmux/claude/adapters/tmux_adapter.sh
```
